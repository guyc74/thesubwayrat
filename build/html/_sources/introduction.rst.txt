Getting started
===============

So, what is a computer game?
A computer game is a computer program that allows YOU to play with it.

So, what is a computer program?
A computer program is a list of commands to the computer.
The commands are written in a language that the computer can understand.
Our game will be written in a language called Python.


Where do we write a program?
----------------------------

The commands are written in one or more documents.
Each document is called a file.

Files are organized in the computer in directories.
Each directory is a folder that contains files
